# v0.3.0 Release Bootstrap and PowerShell Layout
> Status: ACTIVE
> Created: 2026-07-24
> Source: local code-intel-pipeline

## Abstract

Ship `v0.3.0` as the first stable release that can be installed from the bundled Codex Skill
without requiring a source checkout. Pin the default Skill bootstrap to that release, preserve the
public root command surface, move PowerShell contract tests out of the repository root, and archive
completed planning records.

## Core Insight

The repository already contains a checksum-verified GitHub Release bootstrap. The smallest safe
change is to pin its stable default and reorganize non-public scripts without changing the seven
operator commands that existing users invoke.
